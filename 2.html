What is CSS?

Cascading Style Sheets (CSS) is a simple mechanism for addingstyle
(e.g., fonts, colors, spacing) to Web documents
These pages contain information on how to learn and use CSS and on available software.
They also contain news from the CSS working group.

Nearly all browsers nowadayssupport CSS and many otherapplications do, too.
To write CSS, you don't need more than atext editor,
but there are many tools available that make it even easier.
Of course, all software has bugs, even after several updates.
And some programs are further ahead implementing the latest CSS modules than others.
Varioussites describe bugs and work-arounds.
